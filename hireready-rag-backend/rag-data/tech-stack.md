# Interview Preparation Tech Stack Guide

## Frontend Developer
**Recommended Stack:** HTML5, CSS3, JavaScript (ES6+), React.js, Tailwind CSS, TypeScript

**Best For:**
- UI/UX-focused roles
- Startups and product companies
- Companies with modern tech stacks

**Key Skills to Master:**
- Component-based architecture
- State management (Redux, Zustand, Context API)
- Responsive design
- API integration
- Browser DevTools
- Version control (Git)

**Interview Focus:**
- DOM manipulation
- JavaScript fundamentals (closures, promises, async/await)
- React hooks and lifecycle
- Performance optimization
- Accessibility (a11y)
- Cross-browser compatibility

**Salary Range:** ₹4-12 LPA (fresher to 2-3 years)

---

## Backend Developer
**Recommended Stack:** Node.js, Express.js, MongoDB/PostgreSQL, REST APIs, Authentication (JWT)

**Alternative Stack:** Java Spring Boot, Python Django/Flask, .NET Core

**Best For:**
- API development
- Database design
- Scalable systems
- Server-side logic

**Key Skills to Master:**
- RESTful API design
- Database design and queries
- Authentication & authorization
- Security best practices
- Caching (Redis)
- Microservices architecture

**Interview Focus:**
- Data structures and algorithms
- System design basics
- Database normalization
- API security
- Error handling
- Scalability concepts

**Salary Range:** ₹5-14 LPA (fresher to 2-3 years)

---

## Full Stack Developer
**Recommended Stack:** MERN (MongoDB, Express, React, Node.js) or MEAN (Angular instead of React)

**Best For:**
- Startups requiring end-to-end ownership
- Small teams
- Versatile developers

**Key Skills to Master:**
- Both frontend and backend fundamentals
- DevOps basics (Docker, CI/CD)
- Cloud platforms (AWS, Azure, GCP)
- Database management
- API design
- Testing (unit, integration, E2E)

**Interview Focus:**
- Full project lifecycle understanding
- Integration challenges
- Performance optimization
- Security considerations
- Deployment strategies

**Salary Range:** ₹6-16 LPA (fresher to 2-3 years)

---

## Data Science / ML Engineer
**Recommended Stack:** Python, NumPy, Pandas, Scikit-learn, TensorFlow/PyTorch, SQL

**Best For:**
- Analytics roles
- ML/AI product development
- Research-oriented positions

**Key Skills to Master:**
- Statistics and probability
- Machine learning algorithms
- Data preprocessing
- Model evaluation
- Feature engineering
- SQL for data extraction

**Interview Focus:**
- Statistical concepts
- ML algorithms (supervised, unsupervised)
- Model selection and tuning
- Data cleaning techniques
- Python coding challenges

**Salary Range:** ₹6-18 LPA (fresher to 2-3 years)

---

## Confused About Which Path?

### Choose Frontend If You:
✅ Enjoy visual design and user interfaces
✅ Like seeing immediate results
✅ Prefer creative problem-solving
✅ Want faster initial learning curve

### Choose Backend If You:
✅ Enjoy logical thinking and algorithms
✅ Like working with data and databases
✅ Prefer server-side logic
✅ Want to focus on scalability

### Choose Full Stack If You:
✅ Want versatility and complete project ownership
✅ Enjoy both design and logic
✅ Want maximum job opportunities
✅ Like learning diverse technologies

### Choose Data Science If You:
✅ Love mathematics and statistics
✅ Enjoy working with data
✅ Want to build intelligent systems
✅ Interested in AI/ML domain

## Learning Roadmap

### Month 1-2: Fundamentals
- Programming language basics
- Data structures
- Git version control
- Basic algorithms

### Month 3-4: Specialization
- Choose your path
- Learn core frameworks
- Build small projects
- Practice coding daily

### Month 5-6: Advanced & Interview Prep
- System design basics
- Mock interviews
- Portfolio projects
- Open source contributions
- LeetCode/HackerRank practice

**Pro Tip:** Build at least 3 substantial projects showcasing your skills before applying for jobs.
